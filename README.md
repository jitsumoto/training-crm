# Training CRM

職業訓練で作成した Spring Boot 学習用プロジェクトです。  
学生（Student）の登録・一覧・更新・削除を行うシンプルな CRM（顧客管理システム風）アプリケーションです。

---

## 🚀 使用技術

- Java 17
- Spring Boot 3.5.6
  - Spring Web
  - Spring Data JPA
  - Spring Security（現在は無効化設定あり）
  - Thymeleaf（今後拡張予定）
- PostgreSQL
- Maven
- Lombok

---

## ⚙️ 起動方法

```bash
# プロジェクトルートで
mvn spring-boot:run
